Pyxel launcher (commit-SHA CDN)

目的：ランチャーが jsDelivr の commit SHA を含む URL を使って常に最新の web_game.py を読み込みます。GitHub API でコミット SHA を取得し、キャッシュ問題を回避します。API が利用できない場合は raw.githubusercontent へフォールバックします。

注意：このページは GitHub API から最新のコミット SHA を取得し、jsDelivr CDN の URL に @SHA を付与して、エッジキャッシュの古いコンテンツを回避します。SHA は sessionStorage にキャッシュされ、API レート制限を軽減します。
