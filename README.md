## Controls

- **Left/Right Arrow Keys**: Move player
- **Space or Up Arrow**: Jump
- **Q**: Quit (desktop only)

## About

This is a placeholder game featuring:
- A player sprite that can move left and right
- Jump mechanic with gravity physics
- Ground collision detection
- HUD with instructions
- Sound effect on jump
